# 🎙️ VibeVoice AI Studio

A premium, privacy-focused Text-to-Speech (TTS) engine built for creators. No storage, no tracking—just pure performance.

## ✨ Features
- **Ghost-Stream Technology:** Audio is processed in RAM and never touches the disk.
- **Neural Voices:** High-fidelity AI voices for Urdu, Hindi, and English.
- **Dark Mode UI:** Sleek, minimal design for modern developers.
- **Zero Signup:** Open-source and 100% free forever.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **AI Engine:** Edge-TTS
- **Frontend:** Tailwind CSS

## 🚀 Getting Started
1. Clone the repo
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python app.py`
